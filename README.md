# NeuroFill: A SaaS Web App ( on Dev )

## Tech Stack
- Next.js (App Router with Server Actions)
- TypeScript
- Tailwind CSS
- Prisma (PostgreSQL)
- NextAuth (JWT-based authentication)
- Framer Motion (Animations)

## Features
- **Resume Maker**
- **Privacy Policy & Terms Generator**
- **QR Code Generator**
- **Invoice Generator**
